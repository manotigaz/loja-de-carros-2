Classes agora separadas em arquivos individuais dentro da pasta 'classes/':

└── Venda.ts                  → Classe de Vendas


✅ LÓGICA DENTRO DAS CLASSES
==============================

1. CLASSE CARRO (Superclasse)
   - Atributos: modelo, marca, ano, precoCompra, precoVenda, vendido, foto
   - Métodos:
     • vender() - marca como vendido
     • depreciar() - reduz 10% do valor
     • lucro() - calcula lucro
     • getInfo() - retorna informações
     • Getters para todos os atributos

2. CLASSE CARROELETRICO (extends Carro)
   - Atributos adicionais: bateria, autonomia
   - Construtor: recebe parâmetros + chama super()
   - Sobrescreve getInfo() com informações de eletricidade

3. CLASSE CARROCOMBUSTAO (extends Carro)
   - Atributos adicionais: tipoCombustivel, consumo
   - Construtor: recebe parâmetros + chama super()
   - Sobrescreve getInfo() com informações de combustão


✅ ABA DE CLIENTES (PF E PJ)
=============================

Menu de Clientes com 2 tipos:

1. CLIENTE PF (Pessoa Física)
   - Campos: Nome, Endereço, Telefone, Email
   - Específicos: CPF, Data de Nascimento
   - Classe: ClientePF (extends Cliente)

2. CLIENTE PJ (Pessoa Jurídica)
   - Campos: Nome, Endereço, Telefone, Email
   - Específicos: CNPJ, Nome Empresa, Inscrição Estadual
   - Classe: ClientePJ (extends Cliente)

Ambas implementam o método abstrato getInfo()


✅ CLASSE DE VENDA
==================

Classe Venda gerencia:
- ID único (gerado automaticamente)
- Referência ao Carro vendido
- Referência ao Cliente
- Data da venda
- Preço de venda
- Desconto aplicado
- Método getPrecoFinal() para calcular total

Métodos:
• getInfo() - exibe todas as informações da venda
• getters - para acessar dados individuais
